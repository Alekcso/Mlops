# Model Card — Netflix Sentiment Classifier

## 1. Цель модели
Модель предназначена для определения тональности текстовых отзывов о контенте Netflix.  
Поддерживаются два режима инференса:
- **Binary**: positive / negative  
- **3-class**: positive / neutral / negative

Модель используется как демонстрация полного MLOps-пайплайна:
- версия данных,
- конфигурации обучения,
- MLflow-трекинг,
- FastAPI-инференс,
- Docker-упаковка.

---

## 2. Используемые данные
Источник: обучающий набор отзывов Netflix (учебный датасет).  
Использованы две версии препроцессинга:

### **V1 — мягкая очистка**
- удаление HTML, ссылок, спецсимволов,
- лемматизация,
- удаление стоп-слов,
- минимальная фильтрация.

### **V2 — строгая очистка**
- всё как в V1,
- удаление коротких текстов (≤ 3 слов),
- меньше шума → подходит для 3-классовой модели.

---

## 3. Модели и конфигурации

### **Exp1 — Binary (V1, unigrams)**
- max_features: 5000
- ngram_range: (1,1)
- accuracy: **0.8381**
- macro-F1: **0.8380**

### **Exp2 — Binary (V1, bigrams) — лучший бинарный вариант**
- max_features: 10000
- ngram_range: (1,2)
- accuracy: **0.8420**
- macro-F1: **0.8419**

### **Exp3 — 3-class (V2, multinomial)**
- max_features: 5000
- ngram_range: (1,2)
- accuracy: **0.7043**
- macro-F1: **0.6953**

## 5. Ограничения
- Модель работает только для английского языка.  
- Чувствительна к сарказму, сленгу, эмодзи.  
- Не учитывает контекст нескольких предложений.  
- Обучена на домене Netflix — переносимость ограничена.
